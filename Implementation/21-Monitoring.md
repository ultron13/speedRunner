# Monitoring

Monitoring provides real-time insight into active runs and platform health.

## Metrics

- Active runs.
- Active virtual users.
- Transactions per second.
- Response time percentiles.
- Error rate.
- Throughput.
- Pod CPU and memory.
- Redis latency.
- Kubernetes node capacity.
- Jenkins pipeline status.

## Dashboards

- Active test run dashboard.
- Platform operations dashboard.
- Kubernetes execution dashboard.
- Redis runtime data dashboard.
- Application under test dashboard.
- Release quality dashboard.

## Tools

Grafana with Prometheus is recommended for time-series metrics.
